install
iOS 
1- yarn install (npm install)
2 - cd ios
3 - bundle install
4 - pod install 
5 - cd ..
6 - yarn ios (npx run-ios)

install
Android


1- yarn install (npm install)
2- cd android
3- ./gradlew clean
4 - cd ..
5 - yarn android (npx run-android)
